# Delta Mobile UI Kit

A high-fidelity interactive prototype of the Delta mobile app (iOS + Android via Flutter).

## Design Specs
- **Canvas size:** 390 × 844px (iPhone 14 Pro)
- **Dark mode first**
- **Fonts:** Plus Jakarta Sans (display) + Plus Jakarta Sans (body) via Google Fonts
- **Icons:** Lucide Icons (stroke, 1.5px weight)
- **Accent:** #EC4899 pink / #00D4AA live teal / #F59E0B delt gold

## Screens Included
1. **Onboarding** — Welcome + value prop
2. **Discover** — Profile card stack with like/pass controls
3. **Matches & Chat** — Match list + individual chat thread
4. **Live** — Pool entry + live session in progress
5. **Wallet** — delt balance, packs, transaction history
6. **Profile** — Own profile view + edit

## Usage
- Open `index.html` in a browser
- Click/tap nav items and buttons to navigate between screens
- Tweaks panel available for color/mode options

## Component Files
All components are inlined in `index.html` for portability.
